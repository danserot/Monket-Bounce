# Monkey Bounce

**Monkey Bounce** — это простая 2D browser game, где обезьянка прыгает по батутам, собирает бананы и поднимается всё выше.  
Игра сделана на **HTML, CSS и JavaScript** с использованием **Canvas API**.

## Gameplay

Цель игры — подниматься как можно выше, прыгая по платформам, и набирать очки.

В игре есть:
- обычные батуты
- движущиеся батуты
- ломающиеся батуты
- бананы, которые дают дополнительные очки
- система рекорда
- рестарт после проигрыша

## Controls

- **Left Arrow** / **A** — движение влево
- **Right Arrow** / **D** — движение вправо
- **Space** — перезапуск игры после поражения
