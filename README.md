# Monkey Bounce

**Monkey Bounce** - простая 2D-игра в браузере: обезьянка прыгает по батутам, собирает бананы и поднимается как можно выше.

Игра написана на **HTML**, **CSS** и **JavaScript** с использованием **Canvas API**.

## Gameplay

Цель игры - подниматься выше, прыгать по платформам и набирать очки.

В игре есть:
- обычные батуты
- движущиеся батуты
- ломающиеся батуты
- бананы, которые дают дополнительные очки
- система рекорда
- рестарт после проигрыша

## Controls

- **Left Arrow** / **A** - движение влево
- **Right Arrow** / **D** - движение вправо
- **Space** - перезапуск игры после поражения

## Technologies

- **HTML5**
- **CSS3**
- **JavaScript**
- **Canvas API**
